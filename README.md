# Backend Project

A Node.js Express backend with authentication functionality.

## Features

- User registration
- User login
- Password hashing with bcrypt
- JSON file-based user storage

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `node index.js` to start the server on port 3000

## API Endpoints

- POST /api/auth/register
- POST /api/auth/login
